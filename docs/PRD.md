# Product Requirements Document: AtomAssist Live

AtomAssist Live is a browser-based real-time video support platform.

The agent can:
1. Create support sessions.
2. Copy invite links.
3. Join video calls.
4. Chat with customer.
5. End sessions.
6. View history.

The customer can:
1. Join only using a valid invite link.
2. Join from browser.
3. Use audio/video.
4. Chat during the call.
5. Leave the call.

Core requirements:
1. Agent creates session.
2. Customer joins via invite.
3. Both join from browser.
4. Audio/video works in real time.
5. Media must route through our own server.
6. In-call chat works.
7. Chat is persisted.
8. Session history is persisted.
9. Roles are enforced.

Bonus features:
1. Admin dashboard.
2. File sharing.
3. Reconnect handling.
4. Recording.
5. Observability metrics.

Success demo:
1. Agent creates session.
2. Customer joins via link.
3. Both users see and hear each other.
4. Chat works and persists.
5. Agent ends session.
6. History shows events and chat.
